## 📂 프로젝트 구조

```plaintext
src/main/java/com/moodcalendar
├── controller         # REST API
├── service            # 비즈니스 로직
├── mapper             # MyBatis 매퍼 인터페이스
├── domain             # 도메인 모델
├── dto                # Request/Response DTO
├── config             # 설정 파일
└── MoodCalendarApplication.java

src/main/resources
├── mapper/            # MyBatis XML 쿼리
├── static/            # 정적 리소스
├── templates/         # HTML 템플릿 (JSP or Thymeleaf)
└── application.yml

```
